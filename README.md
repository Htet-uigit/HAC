# Hierarchical Agglomerative Clustering (HAC) Student Analytics

##  Project Overview
This project is a **Student Analytics Web Application** built using **Hierarchical Agglomerative Clustering (HAC)**. It analyzes student data and groups them into clusters based on their characteristics and behaviors. The system helps identify patterns among students, supporting better decision-making in education such as personalized learning and performance improvement.

---

##  Features
-  Perform **Hierarchical Agglomerative Clustering**
-  Visualize clusters using **Dendrogram**
-  Reduce dimensions using **PCA (Principal Component Analysis)**
-  Display **cluster characteristics (mean values)**
-  Interactive **Streamlit Web App**
-  Support for **student behavior analysis**
